## 介绍
一个基于[WECHATY](https://github.com/Chatie/wechaty) API 写的一个微信管理工具，因为自己也在做公众号，本来 windows 上有 wetool，但是不能在 mac 上用，就让我想起了做这个项目。

## 安装
```bash
$ git clone https://github.com/crazylxr/wechat-tool.git
$ cd wechat-tool && npm install
& node main.js
```
